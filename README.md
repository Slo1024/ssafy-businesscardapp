# 📇 명함.zip (Business Card Management App)

**명함.zip**은 종이 명함의 한계를 해결하고, 쉽고 편리하게 명함을 관리할 수 있도록 만든 모바일 앱입니다.  
종이 명함의 내구성 문제와 수정 불가능성, 환경적 문제를 해결하며, 스마트폰 하나로 명함을 관리하고 공유할 수 있습니다.

---

## ✨ 주요 기능
- **전화 수신 시 명함 위젯**  
  → 상대방 명함 정보와 직접 남긴 메모 표시, AI 요약 기능 제공
- **명함 공유**  
  → QR 코드 / 링크 / 위젯을 통한 간편한 명함 전달
- **명함 생성 및 관리**  
  → 내 명함 제작, 주고받기(흔들기 기능), 종이 명함 스캔 등록
- **메모 등록**  
  → 명함별 맞춤 메모 작성 및 관리

---

## 🛠 사용 기술
- **Android Studio (Kotlin, Jetpack Compose)**
- **Google ML Kit & Naver Clova OCR** (명함 텍스트 인식)
- **OpenAI API** (명함 텍스트 필드 자동 분류)
